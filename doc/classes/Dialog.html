    buttons

    Array

    Defined in src/Dialog.js:75

    底部按钮组
    当不设置type的时候，按钮的回调直接用callback定义就好，
    如果设置了type为'success'或者'cancel'，callback属性将不会起到作用，此时可以通过dialog提供的promise来注册点击成功或者取消按钮时的回调事件:

    
   Dialog.confirm('确定你已经打开了么？').then(()=>{
           alert('确定');
       },()=>{
           alert('取消');
       });

    按钮组格式如下： 
           
       [
            {
                type: 'success',
                name: '确认'
            },
            {
                type:'cancel',
                egStyle:'white',
                name:'取消'
            },
            {
                egStyle:'warning',
                name:'自定义按钮',
                callback:()=>{alert('自定义按钮');}
            }
        ]

    Default:

    empty
