    ui Module

    Defined in: lib/TooltipPanel.js:53

    eagle-ui部分UI组件
    Crumb封装层级链接(面包屑)
    Slider组件展示幻灯片形式的照片展示
    Label定义标签展示的形式
    Search定义不同的搜索框
    Star以星星的形式定义不同的星级展示
    Calendar为日期选择组件
    Dialog模拟了浏览器弹出的alert、confirm等交互形式
    Paging展示了常见的分页形式标签，封装了点击功能
    Suggestion在Search的基础上，动态联想匹配项
    Tab模拟了常见的导航条样式
    Toast可以模拟Ajax请求成功或者失败后，通知用户的UI界面展示
    Tooltip在Label的基础上，增加了鼠标hover或者点击的提示功能
